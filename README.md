## Install and Usage

1. 원격저장소에서 로컬저장소로 파일을 불러옵니다.

```sh
git clone https://github.com/zhsks528/dot-canvas.git
```

2. 폴더로 이동해서 실행시킵니다

```sh
cd <파일명> // 파일이 저장된 경로로 이동
```

3. npm module을 설치합니다.

```sh
npm install
```

4. 실행시킵니다

```sh
npm run start
```
